<a name="overview"></a>
## Overview

The RP2040 & WizFi360 examples with CircuitPython use **WizFi360-EVB-Pico** - Wi-Fi I/O module built on [**RP2040**][link-rp2040] and WIZnet's [**WizFi360**][link-wizfi360] Wi-Fi module.
